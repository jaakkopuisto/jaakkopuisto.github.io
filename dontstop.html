Don't Stop 3D

The game is set in a futuristic world where the player is a robot that has become self aware, and is now trying to escape this "testing factory" of some sort.

Features

Path Generation

At the moment the path generation is very basic, I've made some world "block" prefabs and those get spawned in a random order, which then loops. Current plan is to make more of those blocks, to easily add more to the experience.

It would be nice to have a randomly generated map with more elements that get randomized, but I think that is outside the scale of this project for this school course.

Player mechanics

Stamina

I've implemented a stamina system to the game, to make the player have something else to worry about as well, other than the map itself.

Currently basic movement such as running and jumping don't cost stamina, while sprinting and wallrunning do.

Wallrun

In the video you see the 0.1 version of the wallrun mechanic. It's simply there to get a visual reference on what the feature will enable the player to do. I plan on making it seem more natural, in the sense that when the player jumps on the wall, the wallrun itself responds to the physics already affecting to the player. For example if you are already going down fast, you simply can't wallrun. Depending on how much speed the player still has going upwards, the longer the wallrun.

Sliding

Not yet implemented in the game, but relatively easy to add. At this state of the game, there might not be too much need for this feature, but I plan on adding more dynamic elements to the map, such as moving traps or other lethal elements, which then could be dodged with the slide for example.



Project Goal

ICT Showroom

The main goal is to have this project ready when we are going to present this project march 5th in ICT Showroom public event in Turku, and should have all the features mentioned above ready by then.
