RTS Project

Features

Camera

This project started from me trying to create a nice "orbit" style camera, with which it would be easy to look around the game view while prototyping. I love playing Total War games, and thought that their camera style would be perfect for this instance.

the camera has automatic height adjustment, which uses similar movement smoothing as the camera rotation. It also uses raycasting to detect objects in it's close proximity, and adjusts height to prevent clipping effects.

Pathfinding

Next I needed to figure out more about how I can create AI for this large scale RTS game. I didn't really have any prior experience working with AI, so I spent quite a lot of time just messing around.

For now I've settled with Unitys own AI pathfinding system, due to the very easy integration to any project. I also didn't want to spend too long figuring out efficient ways of computing AI, because I felt like it would require better knowledge of C#. Also I was very eager on getting other features of the game up and running.

Unit Movement & Input

The game generates unit positions from the mouse position on screen, and spreads them out. I also added some randomness to the position generation, so it would seem more natural.

The game already features basic unit control groups, but I've yet to implement them fully. I'm working on making the camera transition from the current position to the control groups location if the group's hotkey is pressed twice.



Project Goal

Look and feel

I'm a massive Warhammer 40k fan, and I want this project to be set in that fantasy universe.

The game should feel and play like many previous RTS games but I'll also add my own twists to it. Think of Starcraft 2 or Age of empires II in terms of the different unit commands: attack move, patrol, guard etc. Combine that with Company of Heroes style of unit control, with dynamic cover systems. I'd like to make basebuilding a key feature in this game.

Learning

I don't think I'm going to finish this project anytime soon, since RTS games have a lot of things to worry about development wise. I'm not going to worry about that, and just try to make the game I want to make, and learn as much as I can in the process.
